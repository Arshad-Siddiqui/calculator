# Calculator
## Goals

- Create a simple calculator web app.
- Learn to improve props and state structure.
- Improve CSS skills.

## Technologies used

- React (TypeScript, Vite, Vitest)
- HTML
- CSS

## How to run

- Clone the repository
- Run `npm install`
- Run `npm run dev`
- Open `localhost:5173` in your browser

## How to test

- Run `npm run test`
